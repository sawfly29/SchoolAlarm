Программа Школьный звонок v. 1.0

Данная программа предназначена для воспроизведения мелодий с урока и на урок согласно расписанию.

Сама программа по умолчанию находится по пути C:\SchoolAlarm.

Инструкция по установке программы (при неработоспособности программы сразу после установки) находится здесь.

Инструкция по настройке программы, установке расписания звонков и смены мелодий находится вот тут.

Работа программы выглядит как открытое окно командной строки. Если его нет, то на рабочем столе есть папка Шеольный звонок, там можно принудительно запустить программу, или же просто перезапустите компьютер.
