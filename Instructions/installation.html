Инструкция по установке программы Школьный звонок v. 1.0

Установочный файл представляет из себя самораспаковываемый архив. По умолчанию путь установки C:\SchoolAlarm.

После установки автоматически запускается скрипт installer.bat (находится в папке bats). В свою очередь, скрипт осуществляет

  • Работу с планировщиком - добавляет задания по ежедневной перезагрузки компьютера (не столько по причине нестабильности программы, сколько по причине неработоспособности самой ОС Windows) и задание по автозапуску программы Школьного звонка.
  • Запускает установку серверной платформы NodeJS. Это основной движок программы, на его основе происходит функционирование модулей.

Если установка по каким-то причинам не удалась, то необходимо позвать админа. Нужно покомпонентно выяснить правильность установки программы, конкретно - наличие в планировщике заданий правильно созданных задач по перезапуску ОС (опционально) и по запуску программы с помощью NodeJS. За запуск овтечает команда 'node main.js' через cmd при нахождении в папке с программой.

В обычном состоянии работа программы выглядит, как показано на рисунке ниже. Окно, естественно, закрывать не нужно. Основной принцип работы программы просто, а потому и эффективен: раз в минуту программа сверяет часы, и если время совпадает с расписанием, то звенит звонок.

Для удяления программы можно воспользоваться uninstaller.bat, который находится в папке bats (запуск с правами администратора). Ну или вручую по той же схеме: удаление заданий из планировщика задач, удаление платформы NodeJS (наврядли она где-то еще на компьютере пригодится) и удаление папки с самой программой.
